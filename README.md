## 🌐 Webforms-Crud
Exemplo de criação CRUD C# WebForms com banco de dados MySQL.

#### O que voçê vai ver nesse Projeto
| Tecnologia | Descrição |
|-----------|-----------|
| **Blob** | Tipo de dado no MySQL utilizado para armazenar dados binários, como imagens, áudios, vídeos ou documentos PDF|
| **WebForms** | Tecnologia tradicional da Microsoft usada para criar sites e aplicativos web, usando uma lógica de "arrastar e soltar" |

#### Requisitos do Projeto
- Para executar a aplicação é necessário a executar o Script do MySQL.

#### ⚠️ String de conexão do banco
Modifique a string de conexão no arquivo **app.config**, no trecho indicado:

```bash
connectionString="server=127.0.0.1;userid=root;password=SUASENHA;database=webform;persistsecurityinfo=True;"
```
O script para criação da tabela do exemplo encontra-se na pasta **Database**.

#### 🔄 Executar a aplicação
- Necessário varinha mágica na Solução do Visual Studio
